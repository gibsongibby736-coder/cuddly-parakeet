# 🌑 Shadow AI — WhatsApp Bot

بوت واتساب مدعوم بـ Gemini 2.5 Flash، يعمل فقط في المجموعات.

## ⚙️ الإعداد

### 1. افتح server.js وضع مفتاحك:
```js
const GEMINI_API_KEY = 'مفتاحك_هنا';
```

### 2. ارفع على Render:
- أنشئ Web Service جديد
- اربطه بـ GitHub repo يحتوي الملفين
- في Environment Variables أضف:
  ```
  GEMINI_API_KEY = مفتاحك_هنا
  ```
- Start Command: `node server.js`

### 3. عند أول تشغيل:
- ستظهر QR Code في logs الـ Render
- افتح واتساب على الرقم الإضافي
- اضغط النقاط الثلاث ← الأجهزة المرتبطة ← ربط جهاز
- امسح الـ QR Code

### 4. أضف البوت للمجموعة وجربه:
```
!shadow كيف حالك؟
!shadow اشرح لي البرمجة
```
أو أرسل صورة مع كتابة `!shadow` في الوصف

## 📌 طريقة الاستخدام في المجموعة:
| الأمر | الوظيفة |
|-------|---------|
| `!shadow [سؤال]` | سؤال نصي |
| `!shadow` + صورة | تحليل صورة |
| `شادو، [سؤال]` | بالعربية |
| `@البوت [سؤال]` | عبر mention |
